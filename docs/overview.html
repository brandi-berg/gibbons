Overview
Built for recruiting leaders who design like engineers.

Gibbons is a production-grade talent acquisition workspace plugin built to run natively inside Claude Cowork[cite: 3]. Rather than running erratic, context-blind prompt chats, Gibbons executes an unbroken, data-driven pipeline constrained by explicit security gates[cite: 3].

Every single output is rigorously validated against company context files and core configuration parameters[cite: 1, 3]. There is zero inference. If information is missing from a candidate profile, Gibbons outputs an explicit warning rather than making programmatic assumptions[cite: 3].

Version 2.0 introduces deep automation wiring with external platform skills like skillsmate and intelmate, alongside an aggressive /redteam pre-mortem matrix to challenge strategic plan vulnerabilities before outreach occurs[cite: 3].

Candidate Truth Protocol

Every claim, rating, or validation metric must map verbatim to explicit evidence blocks inside a candidate's file. Zero hallucination allowed[cite: 3].

Tight Context Binding

System activities are restricted by local data definitions (Voice/Tone, Ideal Candidate Profiles, Technical Guardrails). Generic output triggers a loop fail[cite: 1, 3].

Platform Skill Orchestration

Natively hooks into automated scraping tools to extract competitor compensation, career architecture, and talent brand scoring on the fly[cite: 3].

Cross-Session Handoffs

Maintains persistent memory directories and structural handoff blocks so context moves securely from phase to phase without duplication of data[cite: 3].
